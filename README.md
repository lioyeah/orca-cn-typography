# OrcaNote 中文排版增强插件

OrcaNote 中文高级排版增强插件，为 [OrcaNote](https://orca-studio.com) (虎鲸笔记) 提供符合[中文排版指南](https://github.com/aaranxu/chinese-copywriting-guidelines)的智能排版优化能力。

## 功能特性

### 核心排版功能

- ✅ **中英文自动空格** - 在中英文/数字之间自动添加空格（如「测试test」→「测试 test」）
- ✅ **标点符号规范化** - 移除多余空格、统一引号样式
- ✅ **引号风格转换** - 支持大陆、港台、技术三种引号风格
- ✅ **增强空格规则** - 数字与单位间自动加空格（如「10GB」→「10 GB」），保留特殊符号（如 233°、15%）
- ✅ **自定义规则** - 支持 JSON 格式的自定义空格和标点规则

### 双模式排版系统

- ✅ **预览模式（Preview Mode）** - 仅视觉显示优化，不修改原文，适合实时查看效果
- ✅ **自动模式（Auto Mode）** - 按 Enter 时自动应用格式化（可撤销），永久修改内容

### 字体与样式设置

- ✅ **全局字体大小** - 自定义基础字体大小（10px-18px）
- ✅ **全局行高** - 设置全局行高（如 1.6、1.8）
- ✅ **正文连字** - 启用字体连字优化西文排版（fi、fl 等）
- ✅ **代码连字** - 可选在代码块中启用连字（默认关闭避免符号混淆）
- ✅ **表格数字对齐** - 使用等宽数字，表格数据更清晰

### 高级功能

- ✅ **输入时暂停** - 打字时暂停排版，停止输入后自动应用，避免干扰
- ✅ **硬格式化到剪贴板** - 将格式化结果复制到剪贴板
- ✅ **硬格式化并写回** - 通过 API 持久化格式化结果，保留富文本格式
- ✅ **调试日志** - 可选启用详细调试信息

## 安装

1. 从 [Releases](../../releases) 下载最新版本
2. 将插件文件夹放入 OrcaNote 插件目录
3. 在 OrcaNote 中启用插件

## 使用方法

### 设置界面

在插件设置中可以配置：

- **排版模式**：选择 Preview 或 Auto 模式
- **自动处理总开关**：开启/关闭实时排版
- **中英文自动空格**：开启后自动在中英文间加空格
- **标点符号规范化**：规范化标点和引号
- **引号风格**：大陆（""『』）、港台（「」『』）、技术风格
- **字体大小和行高**：自定义全局样式
- **连字设置**：正文和代码块的连字选项

### 命令

插件注册了两个命令：

1. **硬格式化到剪贴板** - 将当前选区的格式化结果复制到剪贴板
2. **硬格式化并写回选区** - 将格式化结果永久写回文档（可撤销）

### 自定义规则

支持 JSON 格式的自定义规则：

**自定义空格规则示例：**
```json
[
  {"pattern":"([\\u4e00-\\u9fff])[(]","replacement":"$1 ("},
  {"pattern":"[)]([\\u4e00-\\u9fff])","replacement":") $1"}
]
```

**自定义标点规则示例：**
```json
[
  {"pattern":"[.]{3,}","replacement":"……"},
  {"pattern":"([\\u4e00-\\u9fff])!([\\u4e00-\\u9fff])","replacement":"$1！$2"}
]
```

## 开发

### 环境准备

```bash
# 克隆仓库
git clone https://github.com/lioyeah/orca-cn-typography.git
cd orca-cn-typography

# 安装依赖
npm install
```

### 开发命令

```bash
# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 构建并监视文件变化
npm run build -- --watch
```

### 插件加载到 OrcaNote

1. 构建完成后，将整个项目文件夹复制到 OrcaNote 的插件目录：
   - **Windows**: `%APPDATA%\OrcaNote\plugins\orca-cn-typography`
   - **macOS**: `~/Library/Application Support/OrcaNote/plugins/orca-cn-typography`
   - **Linux**: `~/.config/OrcaNote/plugins/orca-cn-typography`

2. 在 OrcaNote 中打开「设置」→「插件」，启用「orca-cn-typography」

3. 开发时可以使用 `npm run dev` 监视变化，然后重新加载插件即可看到更新

### 技术栈

- **TypeScript** - 主开发语言
- **Vite** - 构建工具
- **OrcaNote Plugin API** - 插件接口

### 项目架构

- `src/main.ts` - 插件入口，处理生命周期和设置
- `dist/index.js` - 构建后的主文件（实际加载到 OrcaNote）
- `PreviewFormatter` 类 - 显示层格式化（DOM 操作）
- `AutoFormatter` 类 - 编辑层格式化（API 调用）

### 调试

在插件设置中启用「调试日志」选项，可以在浏览器控制台查看详细的运行日志。
